2. How We Use Your Information

Primary Purposes

  • Service Delivery: Enable video uploads to YouTube, TikTok, and other platforms

  • Customization: Process video metadata and upload preferences
  • AI Features: Generate video titles using OpenAI API

2.1 Legitimate Interests

  • Improving service performance and reliability
  • Debugging and troubleshooting technical issues
  • Analyzing usage patterns to enhance user experience
  • Preventing abuse and ensuring security

2.2 Legal Compliance

  • Complying with applicable laws and regulations
  • Responding to legal requests and court orders
  • Protecting our rights and the rights of others

3. Data Storage and Security

3.1 Local Storage Approach

Privacy by Design: We minimize data collection by storing most information locally in your browser rather than on our servers.
  • OAuth Tokens: Stored in browser localStorage, never transmitted to our servers
  • Video Files: Processed in browser memory, directly uploaded to target platforms
  • User Preferences: Saved locally, can be cleared by user at any time
  • Session Data: Temporary, cleared when browser session ends

3.2 Security Measures

  • HTTPS Encryption: All data transmission is encrypted using TLS
  • No Server Storage: Sensitive data is not stored on our servers
  • Token Isolation: OAuth tokens are sandboxed within browser origin
  • Secure APIs: Integration with platform APIs using industry standards

5. Your Privacy Rights

5.1 Access and Control

  • Token Management: View, validate, and delete stored OAuth tokens
  • Local Data: Clear browser storage to remove all local data
  • Service Logs: Request information about data we may have collected

5.2 Data Portability

  • Export your OAuth tokens (though we recommend secure handling)
  • Download any configuration data you've created
  • Transfer data between browser instances

5.3 Right to Deletion

  • Clear all locally stored data through browser settings
  • Request deletion of any server-side logs or analytics data
  • Discontinue service use at any time

6. Cookies and Tracking

6.1 Essential Cookies

  • Session Management: Maintain application state during use
  • Authentication: Remember OAuth token status
  • Preferences: Store user interface preferences

6.2 Analytics Cookies

  • Usage Analytics: Understand how features are used

Cookie Control: You can disable cookies through your browser settings, but this may affect service functionality.

7. Data Retention

7.1 Local Data

  • Stored indefinitely in your browser until you clear it

7.2 Server-Side Data

  • Analytics Data: Aggregated data retained for 2 years
